REST API

HyperKitty comes with a small REST API allowing you to programatically retrieve messages and information.

Formats

This REST API can return the information into several formats. The default format is html to allow human readibility.
To change the format, just add ?format=<FORMAT> to the url

The list of available formats is:

  • json (?format=json)
  • json-p (?format=json-p)
  • txt (?format=txt)
  • xml (?format=xml)
  • html (?format=html)
  • xhtml (?format=xhtml)

Lists /api/list/

Using the address /api/list/ you will be able to retrieve the information known about all the mailing-lists.

For example: {% url 'api_lists' %}

List /api/list/<list name>/

Using the address /api/list//<list name>/ you will be able to retrieve the information known about a specific mailing-lists including latest activity.

For example: {% url 'api_list' mlist_fqdn='devel@fp.o' %}

Message /api/list/<list name>/message/<Message-ID>/

Using the address /api/list/<list name>/message/<Message-ID>/ you will be able to retrieve the information known about a specific message on the specified mailing-list.

For example: {% url 'api_message' mlist_fqdn='devel@fp.o' messageid='13129854572893334' %}

Thread /api/list/<list name>/thread/<ThreadID>/

Using the address /api/list/<list name>/thread/<Message-ID>/ you will be able to retrieve the all the messages for a specific thread on the specified mailing-list.

For example: {% url 'api_thread' mlist_fqdn='devel@fp.o' threadid='13129854572893334' %}

Search /api/list/<list name>/search/<field>/<keyword>/

Using the address /api/list/<list name>/search/<field>/<keyword>/ you will be able to search for all messages of the specified mailing-list containing the provided keyword in the given field.

The list of available field is:

  • From
  • Subject
  • Content
  • SubjectContent

For example: {% url 'api_search' mlist_fqdn='devel@fp.o' field='From' keyword='pingoured' %}
